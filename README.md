# Pokédex Simples

Um simples Pokédex desenvolvido em HTML, CSS e JS, utilizando a API PokeAPI para buscar informações sobre Pokémon e exibi-las em um modal.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript
